# Issue 28: Implementação de Suíte de Testes Automatizados

## Descrição
Estabelecer uma base de testes automatizados para garantir a estabilidade das lógicas de negócio e dos fluxos críticos da aplicação.

## Requisitos
- [ ] **Unit Tests**: Implementar testes para funções de cálculo de NPS, Roda da Vida e progresso de metas.
- [ ] **E2E Tests**: Criar fluxos de teste de ponta a ponta para login, criação de PDI e avaliação de sessão (Cypress/Playwright).

## Critérios de Aceite
- As lógicas de cálculo devem possuir 100% de cobertura de testes unitários.
- Os fluxos críticos devem ser validados automaticamente antes de cada deploy.
